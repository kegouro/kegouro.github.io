## Estructura

```
index.html          La landing — un solo archivo, con toggle de idioma ES/EN
assets/
  pharos.svg        Logo: el haz del faro segmentado en elementos diferenciales
  pharos-mono.svg   Logo monocromático (sellos, una tinta)
  favicon.svg       Favicon
  banner.png        Banner horizontal "By JLB"
  og.png            Tarjeta social (Open Graph, 1200×630)
brand/
  BRAND.md          Manual de marca: paleta, tipografías, logo, firma, watermark
```

## Desarrollo

No hay build. Editas `index.html` y listo.

```bash
python3 -m http.server 8000   # → http://localhost:8000
```

## Marca

Paleta cálida sobre negro (ámbar `#f5a72c`, brasa `#ff7a3c`), tipografías **Fraunces** + **IBM Plex Mono**,
y el logo del haz segmentado. Todo en **[`brand/BRAND.md`](brand/BRAND.md)**.
